# 🛹 Trick Tracker (MVP-1)

> **Проект для райдеров от райдера.**  
> Хватит записывать трюки в заметки или держать их в голове. Trick Tracker — это твой цифровой журнал прогресса, спот-чек и база знаний в одном кармане.

## 🤘 О чем этот проект?
Это бэкенд на микросервисной архитектуре, созданный для того, чтобы ты мог:
*   **Log your progress:** Отмечай, какие трюки ты уже "увез", а какие еще в процессе (Land it vs. Battle).
*   **Trick Dictionary:** Глобальная база скейтерских трюков с категориями (Flat, Grinds, Slides, Vert).
*   **Spot Map:** База спотов с описанием покрытия и "охраняемости".
*   **Skate Profile:** Твоя карточка райдера со статистикой.

## 🏗 Технологический стек
Проект построен на актуальном стеке Java-разработки:
*   **Java 17** — база.
*   **Spring Boot 3.3.x** — каркас приложения.
*   **PostgreSQL** — надежное хранилище твоих профайлов и трюков.
*   **Maven** — сборка и управление модулями.
*   **Docker** (в планах) — для быстрого деплоя.

## 🛠 Структура сервисов
Проект разбит на модули (микросервисы) для удобного масштабирования:
1.  **auth-service** — регистрация и безопасность.
2.  **user-service** — управление профилями райдеров.
3.  **trick-service** — ядро системы: база трюков и прогресс.
4.  **spot-service** — карта и описание спотов.

## 🚀 Как запустить локально

1. **Клонируй репозиторий:**
   ```bash
   git clone https://github.com
   ```

2. **Собери проект через Maven Wrapper:**
   ```bash
   ./mvnw clean install
   ```

3. **Настрой БД:**
   Создай базу данных PostgreSQL и укажи доступы в `application.yml` нужного сервиса.

4. **Запуск:**
   Запусти `Application.java` в интересующем тебя модуле.
